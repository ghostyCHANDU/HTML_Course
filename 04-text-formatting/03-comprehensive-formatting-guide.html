Complete HTML Formatting Tags Reference

This page demonstrates all major HTML formatting tags with live examples.

Text Formatting Tags

<b>

Bold text (presentational)

This is bold text using the b tag.
<strong>

Strong importance (semantic bold)

This is strong text with semantic importance.
<i>

Italic text (presentational)

This is italic text using the i tag.
<em>

Emphasized text (semantic italic)

This is emphasized text with semantic meaning.
<u>

Underlined text

This is underlined text.
<s>

Strikethrough text (no longer accurate)

This is strikethrough text.
<del>

Deleted text (semantic strikethrough)

This is deleted text that was removed.
<ins>

Inserted text (typically underlined)

This is inserted text that was added.
<mark>

Highlighted/marked text

This is highlighted text for reference.
<small>

Smaller text

This is normal text and this is smaller text.
<big>

Bigger text (deprecated in HTML5, but still works)

This is normal text and this is bigger text.

Subscript and Superscript

<sub>

Subscript text

H2O is the chemical formula for water.
<sup>

Superscript text

E = mc2 is Einstein's famous equation.

Code and Computer Text

<code>

Inline code

Use the console.log() function to print to the console.
<pre>

Preformatted text (preserves spaces and line breaks)

function greet() {
    console.log("Hello World!");
    return true;
}
<kbd>

Keyboard input

Press Ctrl + C to copy.
<samp>

Sample output from a computer program

The program returned: Error: File not found
<var>

Variable in mathematical or programming context

The formula is y = mx + b.

Quotations and Citations

<q>

Short inline quotation

He said, To be or not to be, that is the question.
<blockquote>

Block-level quotation

This is a longer quotation that spans multiple lines and is typically displayed as an indented block of text separate from the surrounding content.
<cite>

Citation or reference to a creative work

My favorite book is To Kill a Mockingbird by Harper Lee.
<abbr>

Abbreviation or acronym

The WHO is a specialized agency.
<dfn>

Definition term

HTML is the standard markup language for creating web pages.

Directional and Language Tags

<bdo>

Bi-directional override

Normal text: Hello World
Reversed: Hello World
<bdi>

Bi-directional isolation

User إيان scored 90 points.

Ruby Annotations (East Asian Typography)

<ruby>, <rt>, <rp>

Ruby annotations for showing pronunciation

漢 (かん) 字 (じ)

Time and Progress

<time>

Machine-readable time/date

The meeting is scheduled for February 7, 2026 at 2:00 PM.
<progress>

Progress indicator

Download progress: 70%
<meter>

Scalar measurement within a known range

Disk usage: 60%

Semantic Tags

<address>

Contact information

Written by John Doe
Email: john@example.com
123 Main Street, City, State
<details> and <summary>

Collapsible disclosure widget

Click to expand This content is hidden by default and can be toggled by clicking the summary.

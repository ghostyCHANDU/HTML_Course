# HTML Learning Guide

A structured guide to progressively master HTML using the files in this course.

## Week 1: HTML Fundamentals

### Day 1-2: Getting Started
- **Open:** `my first html.html`
- **Learn:** Basic HTML document structure
- **Concepts:** DOCTYPE, html, head, body tags

### Day 3-4: Understanding Headings
- **Open:** `head.html`
- **Learn:** Heading hierarchy (h1-h6)
- **Concepts:** When to use which heading level
- **Follow-up:** `head7.2.html`, `head7.4.html`

### Day 5: Document Structure
- **Open:** `index1.1..html`, `index1.2.html`, `index1.3.html`
- **Learn:** Paragraphs, basic formatting
- **Practice:** Create your own simple page

## Week 2: Text & Formatting

### Day 1-2: Formatting Tags
- **Open:** `format all tags.html`
- **Learn:** Bold, italic, underline, strong, emphasis
- **Open:** `html-formatting-tags.html`
- **Practice:** Identify when to use which tag

### Day 3-4: Text Styling
- **Open:** `text6.1.html` → `text6.5.html` (progressive complexity)

### Day 5: Paragraphs & Text

## Week 3: Colors

## Learning Tips

### 1. Hands-On Practice
- Don't just read; open each file and examine the code
- Modify values and observe the changes
- Experiment with combinations

### 2. Progressive Complexity
- Master each concept before moving forward
- Review previous files when needed
- Build layer by layer

### 3. Reference & Review
- Keep a text editor open alongside your browser
- Compare example files to understand variations
- Take notes on key concepts

### 4. Document Your Learning
- Create your own examples for each concept
- Write comments explaining how things work
- Build a personal reference library

### 5. Debugging
- Use browser developer tools (F12)
- Check for typos in tags and attributes
- Validate your HTML at: https://validator.w3.org/

## Common Questions

### Q: What browser should I use?
A: Any modern browser works. Chrome, Firefox, and Safari are recommended.

### Q: Can I modify the files?
A: Yes! Modify, experiment, and learn by doing.

### Q: What if I don't understand something?
A: Review that section again, check the reference links, and compare with related files.

### Q: How long does this take?
A: Typically 7-8 weeks at a comfortable pace, but adapt to your speed.

### Q: Where can I go from here?
A: Learn CSS for styling, then JavaScript for interactivity.

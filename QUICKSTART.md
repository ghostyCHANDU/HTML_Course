# Quick Start Guide

Get up and running with the HTML Course in 5 minutes!

## 1. Clone or Download

**Option A: Using Git**
```bash
git clone https://github.com/chandu23-3/HTML_Course.git
cd HTML_Course
```

**Option B: Download ZIP**
- Click "Code" → "Download ZIP"
- Extract the downloaded file

## 2. Open Your First File

### Using a Browser:
1. Locate `my first html.html`
2. Right-click and select "Open with" → Choose your browser
3. Or double-click the file

### Using VS Code (Recommended):
1. Open VS Code
2. File → Open Folder → Select the `HTML_Course` folder
3. Click on `my first html.html`
4. Right-click → "Open in Default Browser"

### Using Python Server (Advanced):
```bash
cd HTML_Course
python -m http.server 8000
```
Then visit: `http://localhost:8000` in your browser

## Troubleshooting

### File won't open
- Make sure you're using the correct browser
- Check file path and spelling
- Try right-clicking → Open with specific browser

### Styling not showing
- Browser may be caching old version
- Do a hard refresh (Ctrl+Shift+R)
- Check browser developer tools (F12)

### Can't see file changes
- Save the file in your text editor
- Refresh the browser page
